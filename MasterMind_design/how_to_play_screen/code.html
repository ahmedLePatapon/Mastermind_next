How to Play Mastermind

Objective

The codebreaker tries to guess the secret code in the fewest number of turns. The codemaker provides feedback on each guess.

Gameplay

  1. The codemaker (the computer) sets a secret code of 4 colored pegs.
  2. The codebreaker (you!) makes a guess by placing colored pegs in a row.
  3. The codemaker provides feedback using key pegs for each guess.

Correct Color, Wrong Position

An empty peg with a white border indicates a correct color but in the wrong position.

Winning and Losing

  • Win: Guess the secret code correctly within 10 turns.
  • Lose: Fail to guess the code within 10 turns.
